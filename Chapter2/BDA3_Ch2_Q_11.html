The posterior density is proportional to the product of unnormalised Cauchy densities.

$p(\theta | y_{1:n}) \propto \prod_{i=1}^{n}\big(1 + (y_i -\theta)^2\big)^{-1}$
¶

a. Compute the unnormalised density on a grid of points¶

A grid approximation plots the unnormalised density as a function of the parameter, the data is fixed.

In [4]:
N_grid_items <- 10000 # How many segments to divide the grid interval
theta_min    <- 0     # Grid minimum
theta_max    <- 100   # Grid maximum

xx           <- seq(theta_min, theta_max, length.out = N_grid_items) # create the grid
unif_d       <- 1/(theta_max-theta_min) # Uniform density

# Calculate approximate posterior
grid_approx  <- sapply(X = xx, FUN = function(x) prod(dcauchy(x=y,location = x,scale = 1)) ) * unif_d

b. Sample the model¶

In [7]:
m<-"
data{

  int<lower=0>  N;                    // Number of observations
  real<lower=0> scale;                // Cauchy scale
  vector[N]     y;                    // Observations

}
parameters{

  real<lower=0, upper=100> theta;     // Cauchy location parameter

}
model{

  // theta ~ uniform(0,100);          (stated in the parameters, definition is redundant here)
  y ~ cauchy(theta, scale);           // Define observational model

}
generated quantities{

  vector[N] y_tilde;
  for(i in 1:N){
    y_tilde[i] = cauchy_rng(theta, scale); // Generate predictions by pushing posterior through observational dist
  }

}
"

m<-"
data{

  int<lower=0>  N;                    // Number of observations
  vector[N]     y;                    // Observations

}
parameters{

  real<lower=0, upper=100> theta;     // Normal location parameter
  real<lower=0> sigma;

}
model{

  // theta ~ uniform(0,100); (stated in the parameters, definition is redundant here)
  sigma ~ std_normal();
  y ~ normal(theta, sigma);         // Define observational model

}
generated quantities{

  vector[N] y_tilde;
  for(i in 1:N){
    y_tilde[i] = normal_rng(theta, sigma); // Generate predictions by pushing posterior through observational dist
  }

}
"
